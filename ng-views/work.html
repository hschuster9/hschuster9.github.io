Tennis Matcher

Web application designed to help tennis players connect/communicate with each other

  • A user can create a post to set up an event (tournament, match, practice, etc.) and other users can comment on that post to express interest.

Technologies:

  • Ruby on Rails
  • PostgreSQL
  • ActiveRecord
  • HTML/CSS
  • Deployed on Heroku
  • Three model association & user authentication
http://tennis-matcher.herokuapp.com
